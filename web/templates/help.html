Помощь

Ниже будут приведены картинки с расположением чекера для каждой лабораторной.
Нужно расположить чекер в соответсвии со схемой.
На экране входа чекера будет написан его IP-адрес из внутренней сети, получаемый по DHCP.
С любой из соседних машин с гарфическим интерфесом нужно зайти в браузер и перейти по адресу http://CHECKER_IP/, где CHECKER_IP - адрес чекера с экрана входа.

Внимание!
Для успешной проверки хостов с ОС Linux или PfSense нужно включить службу SSH.

Как включить SSH на PfSense

В верхней панели System -> Advanced.
Листаем ниже и находим раздел Secure Shell, ставим галочку (Enable Secure Shell) напротив Secure Shell Server.
Затем сохраняем (Save).

Как включить SSH на Linux

Редактируем конфиг: nano /etc/ssh/sshd_config
Меняем строку #PermitRootLogin prohibit-password на PermitRootLogin yes
Выполняем команду: systemctl restart ssh
